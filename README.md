# hcmut-ssps
A smart printing service for students at HCMUT

## Installation

Use the package manager to install packages.

```bash
npm install
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
